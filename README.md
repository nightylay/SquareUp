# SquareUp Website

SquareUp is a multi-page website for an IT company specializing in comprehensive digital solutions.

## Stack

- HTML5
- SCSS
- JavaScript

## Getting Started

1. Clone the repository

```
git clone https://github.com/nightylay/SquareUp.git
cd SquareUp
```

2. Install dependencies

```
npm install
```

3. Run project

```
npm start
```
